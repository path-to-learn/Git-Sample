
Some common git commands

--handy commands
> git init
> git status

-- cloning, This will clone your project from web interface to local machine
> git clone https://github.com/prashant0810/Git-Sample
> git add --all

-- commit
> git add .
> git pull -- uptodates the current project from repositry
> git add --all -- adds files prior to commit
> git commit -m "Message for commit" -- actually commit commands
> git push -- pushes to web ui
